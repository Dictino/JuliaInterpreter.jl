# Installation
```
Pkg.clone("https://github.com/Keno/DebuggerFramework.jl")
Pkg.clone("https://github.com/Keno/ASTInterpreter2.jl")
using ASTInterpreter2
```
Then use the `@enter` macro as on older versions of ASTInterpreter.
